О проекте

MySQL Exercises был разработан Ююкиным Иваном для помощи в освоении SQL запросов на базе данных MySQL. Задачи представленные на сайте были предложенны студентам 3 курса в рамках изучении дисциплины 'Базы данных' университета ТУСУР по программе 'Информатика и вычислительная техника.'

Использованные технологии:
  • MySQL
  • Java 8
  • Spring Boot
  • Spring Security
  • Hibernate
  • JavaScript
  • Thymeleaf
  • Bootstrap
